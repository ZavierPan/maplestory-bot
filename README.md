# maplestory-bot

主要練習寫腳本，語言使用python3，OS為win10

* 計畫

    1. 切換至遊戲視窗
    2. 控制腳色
    3. 定位輪的位置
    4. 解輪
    5. 完整的腳本撰寫

* 可能用到的套件
    ```
    winapi
    pyautogui
    pydirectinput
    opencv
    ```


## Requirement

```
conda create --name maplestory python=3.7

conda activate maplestory
pip install pyautogui

```
